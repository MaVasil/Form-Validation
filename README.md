# 📝 Dynamic Registration Form

A fully interactive **registration form** built with **HTML, CSS, and JavaScript**.  
The form validates user input dynamically using **regular expressions (regex)** and provides **real-time feedback**.

---

## ✅ Features

- **Username Validation:** 3–15 characters; letters, numbers, or underscores  
- **Email Validation:** standard email format  
- **Password Validation:** minimum 8 characters, must include uppercase, lowercase, number, and special character  
- **Confirm Password Validation:** checks in real-time that it matches the password  
- **Real-Time Feedback:** green ✅ for valid input, red ❌ for invalid input  
- **Dynamic User Table:** displays registered users in a structured table below the form

---

## 🛠️ Technologies Used

- **HTML5** – Form structure and layout  
- **CSS3** – Styling and feedback indicators  
- **JavaScript** – Dynamic validations and table updates

---

## ▶️ How It Works

1. Users fill in the form fields (username, email, password, confirm password).  
2. Regex validations check each input in real-time.  
3. Valid fields show a green checkmark ✅; invalid fields show a red warning ❌.  
4. On successful submission, user details are added to the table dynamically.  
5. Form resets for the next entry.
